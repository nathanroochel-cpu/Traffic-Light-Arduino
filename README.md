Traffic Light
A basic traffic light simulation using three LEDs. Cycles through green, yellow, and red with realistic timing.
How it works
Loops continuously:
Green on for 3 seconds
Yellow on for 1 second
Red on for 3 seconds
Hardware
Arduino Uno
Green LED
Yellow LED
Red LED
3x 220Ω resistors
Breadboard + jumper wires
Wiring
```
Green LED  → A2 (with 220Ω resistor to GND)
Yellow LED → A3 (with 220Ω resistor to GND)
Red LED    → A4 (with 220Ω resistor to GND)
```
Setup
Open `TrafficLight.ino` in the Arduino IDE
Select your board and port
Upload and watch it go
